Hypertext Markup Language

Primero conoceremos unas características de este lenguaje:

  • Paginas HTML son documentos de texto

  • HTML usa elementos para definir la estructura de la página

  • Elementos se definen por etiquetas encerradas en paréntesis angulares (< y >)

  • El contenido de un elemento se encierra entre etiquetas

  • Existen etiquetas de apertura y de cierre

    El nombre de la etiqueta debe ser el mismo y se distingue a la etiqueta de cierre con una diagonal

  • Las etiquetas pueden tener atributos los cuales se especifican en una etiqueta de inicio

  • Los atributos contienen información adicional acerca del contenido

<etiqueta atributo="valor"> contenido </etiqueta>

Etiquetas mas comunes

La etiqueta es importante para el elemento que queremos poner, no es la misma etiqueta para un párrafo que la que se usa para un titulo, por lo que tenemos que aprendernos al menos las más básicas

  • <h1></h1>

    Esta etiqueta es usada para indicar que el contenido será mostrado con énfasis, es usado más para títulos y no solo es h1, sino que también puede ser h2, h3, h4, h5 y h6 que van enumerados de mayor a menor importancia.

  • <p></p>

    Esta etiqueta es usada para indicar que el contenido será tratado como un párrafo

  • <br>

    Esta etiqueta es usada para indicar que se va a dar un salto de línea.
    NOTA: esta etiqueta no tiene cierre

  • <b></b>

    Esta etiqueta es usada para indicar que el contenido será mostrado en negritas

  • <i></i>

    Esta etiqueta es usada para indicar que el contenido será mostrado en cursiva

  • <strong></strong>

    Esta etiqueta es usada para indicar que el contenido será mostrado como negritas, pero no solo eso, sino que un buscador lo tomara como palabra clave de la página, así que úsala con sabiduría.

  • <em></em>

    Esta etiqueta es usada para indicar que el contenido será mostrado en cursivas, pero al igual que strong, servirá al navegador como contenido clave para identificar la página

  • <span></span>

    Esta etiqueta es usada para agrupar o seleccionar un texto o una informacion, no será visible por el momento, pero cuando veamos los siguietnes temas veremos cómo nos puede servir.

También hay que agregar que las etiquetas pueden ir una dentro de otra, por ejemplo que un párrafo tenga una o varias palabras o frases en negritas y/o cursivas, o resaltadas unas partes etc. etc., de aquí solo queda que experimentes
