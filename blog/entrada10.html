JQuery

"write less, do more"

Fue escito por John Resig en 2006 y es un script que ya trae muchas de las tareas realizadas comunmente por quien desarrolla una pagina web

Simplifica

  • Interaccion con HTML
  • Manipulacion de DOM
  • Manipulacion de CSS
  • Manejo de eventos
  • Animaciones
  • Interacciones AJAX
  • Garantiza la compatibilidad entre navegadores

Todo se realiza a travez de una funcion llamada [jQuery()] que resibe como parametro una expresion CSS y regresa un objeto jQuery

var $objeto = jQuery(<selector css>)
