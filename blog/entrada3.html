Estructura Semantica de un documento HTML

Antiguamente en verciones anteriores de html, solo existia la etiqueta div para intentar acomodar todos los elementos dentro de una pagina web, tenias que jugar con los divs, cambiar comportamientos comportamientos, delimintarlos, para que pudieras crear una pagina web agradable a la vista, pero con el tiempo se fue notando una uniformidad en la paginas y se opto por poner ciertos espacios estandar en los cuales mostrar la informacion y de esa manera ayudar a facilitar el diseño y sobretodo al buscador, que que estas etiquetas sirven de ayuda al buscador para encontrar mas facilmente la informacion que le solicitan.

Estas son las etiquetas utilizadas para darle una estructura a la pagina web

  1. Encabezado

    header

  2. Navegacion

    nav

  3. Contenido Principal

    main

  4. Seccion

    section

  5. Articulo

    article

  6. Barra lateral

    aside

  7. Pie de pagina

    footer

  8. Diviciones

    div
